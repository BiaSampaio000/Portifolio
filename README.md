# Portifolio
Um portfólio moderno e responsivo criado para apresentar minha trajetória, habilidades em desenvolvimento de software e os serviços que ofereço, com um design elegante e profissional.
Este projeto foi desenvolvido como meu portfólio profissional, com o objetivo de apresentar minha trajetória acadêmica e profissional na área de tecnologia. O site reúne informações sobre meus serviços, projetos desenvolvidos, habilidades técnicas e depoimentos, utilizando um design moderno, responsivo e com identidade visual própria.
